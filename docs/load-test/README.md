# Résultats du test de charge — activité 8

Ces fichiers ont été générés par Locust avec 10 utilisateurs virtuels, un taux
de démarrage de 2 utilisateurs par seconde et une durée de 20 secondes.

Le fichier `labo7_results.html` contient les statistiques finales de 894
requêtes. Le CSV principal a été généré quelques instants avant l'arrêt complet
du test et peut donc afficher un total légèrement inférieur.
